Установка базы данных: sudo apt update sudo apt install -y mongo
Путь к базе данных /var/lib/mongodb Путь к конфигурации /etc/mongodb.conf Если вам нужна другой путь к файлом запускайте со флагом --dbpath
Подключение к MongoDB осуществляет по url = 'mongodb://localhost:27017/myproject' mongodb:// сам протокол localhost:27017 сервер /myproject это название вашей базы данных
sudo mongoexport --db newdb -c restaurants --out newdbexport.json newbd название вашей базы данный restaurants - название коллекции newdbexport.json - название вашего json файла
sudo mongoimport --db newdb --collection restaurants --file primer-dataset.json
sudo systemctl status mongodb проверить состояние службы
sudo systemctl stop mongodb остановить
sudo systemctl restart mongodb Для рестарта
sudo systemctl disable mongodb По умолчанию MongoDB настроена для автоматического запуска вместе с сервером. Если вы хотите отключить автоматический запуск
sudo systemctl enable mongodb Чтобы автоматический запуск восстановить
запускаем в консоле: $ mongo создание базы данных $ use todo вы увидите switched to db todolist, это значит, что база данных подключена далее мы создаем коллекцию в этой базе данных $ db.createCollection('todolist') вставляем запись db.todolist.insert({number: 1}) Чтобы посмотреть все созданные коллекции $ show collections для удаления коллекция $ db.todolist.drop() для удаления всей базы данных $ db.dropDatabase()
В одной базе данных может быть множество коллекций
