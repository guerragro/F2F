Установка базы данных: sudo apt update sudo apt install -y mongo
Путь к базе данных var/lib/mongodb Если вам нужна другой путь к файлом запускайте со флагом --dbpath
sudo systemctl status mongodb проверить состояние службы
sudo systemctl stop mongodb остановить
sudo systemctl restart mongodb Для рестарта
sudo systemctl disable mongodb По умолчанию MongoDB настроена для автоматического запуска вместе с сервером. Если вы хотите отключить автоматический запуск
sudo systemctl enable mongodb Чтобы автоматический запуск восстановить
запускаем в консоле: $ mongo создание базы данных $ use todolist вы увидите switched to db todolist, это значит, что база данных подключена вставляем запись db.todolist.insert({number: 1})
