Для начало нам нужно скомпилировать приложения, делается это с помощью команды "Angular-CLI" (@angle/cli): ng build --prod Содержимое сохранится в корне вашего приложение в папке dist
Следующая часть установка сервера Nginx
Для Mac OSX $ brew install nginx $ sudo nginx
Для Linux $ sudo apt-get install nginx
Для запуска\остановки\рестарта используются следующие команды $ sudo service nginx start $ sudo service nginx stop $ sudo service nginx restart $ sudo service nginx status После каждого обновления nginx.conf делайте restart
Для Mac OSX /usr/local/etc/nginx/nginx.conf
Для Linux /etc/nginx/nginx.conf
Теперь отредактируем файл: http { ... server { listen 8080; server_name localhost; location / { root /var/www/angular-deploy; index index.html index.htm; }
Чтобы ваш ресурс был доступен не только на вашем компьютере, нужно открыть порт 443 sudo iptables-save
