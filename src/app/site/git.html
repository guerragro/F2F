Работа со сташем
git stash save 'Введите название' Сохранение ваших изменений в хранилище
git stash list Показывает ваше хранилище структура: stash@{0}: On ya/hms100 (ваша ветка, она добавляется автоматически): ya/hms100 (это ваше название) stash@{1}: On ya/hms113: hms113 stash@{2}: On ya/hms98: тест
git stash apply stash@{0} Применения одного из них, но в хранилище оно остается
git stash pop stash@{0} Применение и удаление из хранилища
git stash drop stash@{0} Удаление
