Но если, это данные не будут менять, то и не стоит использовать лишний раз get and set service.ts public _wares: any; get wares() { return _this.wares }; set wares(value) { this._wares = value } если в запросе с вашего сервиса у вас приходит какая-то информация, допустим товар, делайте обязательно проверки wares. wares.length !== 0, проверяем на длину массив wares instanceof Array, проверям пришел ли нам массив
